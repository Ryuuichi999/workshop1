HTML Image Element Tutorial

The <img> element is used to embed images into a webpage. Below are examples and explanations of how to use it.

1. Basic Image Example

Hereâ€™s how to add a simple image:

<img src="https://via.placeholder.com/150" alt="Placeholder Image" />

2. Image with Width and Height

You can specify the size of the image using the width and height attributes:

<img src="https://via.placeholder.com/150" alt="Placeholder Image" width="100" height="100" />

3. Importance of the Alt Attribute

The alt attribute provides alternative text for the image, which is displayed if the image fails to load or for screen readers.

<img src="broken-link.jpg" alt="This image could not load" />

4. Responsive Image Example

Using CSS, you can make an image responsive by setting its max-width to 100%:

<img src="https://via.placeholder.com/150" alt="Responsive Image" style="max-width: 100%;" />
