How to Link an External CSS File

Steps to Link CSS

  1. Create a file called style.css.
  2. Add your CSS rules inside style.css.
  3. Use the <link> tag in the <head> section of your HTML to connect it, like this:
    <link rel="stylesheet" href="style.css" />
