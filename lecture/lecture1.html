What Makes a Website?

A website is created using three key technologies: HTML for structure, CSS for styling, and JavaScript for interactivity.

1. HTML (Structure)

HTML defines the structure of a webpage. For example:

<h1>Hello, World!</h1>

Displays a title on your webpage.

2. CSS (Styling)

CSS adds styles to make the page look beautiful. For example:

h1 { "color: blue; font-size: 24px;" }

This makes the text blue and larger.
